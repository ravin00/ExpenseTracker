# ExpenseTracker Design System

## Color Palette

### Primary Colors
- **Primary Blue**: #3B82F6 (Main actions, navigation)
- **Primary Dark**: #1D4ED8 (Hover states)
- **Primary Light**: #DBEAFE (Backgrounds)

### Semantic Colors
- **Success**: #10B981 (Income, positive trends)
- **Warning**: #F59E0B (Budget warnings)
- **Danger**: #EF4444 (Overspending, expenses)
- **Info**: #06B6D4 (Analytics, insights)

### Neutral Colors
- **Gray 50**: #F9FAFB (Page background)
- **Gray 100**: #F3F4F6 (Card backgrounds)
- **Gray 200**: #E5E7EB (Borders)
- **Gray 400**: #9CA3AF (Placeholder text)
- **Gray 600**: #4B5563 (Secondary text)
- **Gray 900**: #111827 (Primary text)

## Typography

### Font Family
- **Primary**: Inter (Clean, modern, readable)
- **Fallback**: system-ui, -apple-system, sans-serif

### Font Sizes
- **Text XS**: 12px (Captions, helper text)
- **Text SM**: 14px (Body text, labels)
- **Text Base**: 16px (Main content)
- **Text LG**: 18px (Subheadings)
- **Text XL**: 20px (Card titles)
- **Text 2XL**: 24px (Page titles)
- **Text 3XL**: 30px (Dashboard headers)

### Font Weights
- **Light**: 300 (Subtitles)
- **Regular**: 400 (Body text)
- **Medium**: 500 (Labels, buttons)
- **Semibold**: 600 (Headings)
- **Bold**: 700 (Important headings)

## Spacing System
- **XS**: 4px
- **SM**: 8px
- **MD**: 16px
- **LG**: 24px
- **XL**: 32px
- **2XL**: 48px
- **3XL**: 64px

## Border Radius
- **SM**: 4px (Small elements)
- **MD**: 8px (Cards, buttons)
- **LG**: 12px (Modals)
- **XL**: 16px (Hero sections)
- **Full**: 9999px (Pills, avatars)

## Shadow System
- **SM**: 0 1px 2px 0 rgba(0, 0, 0, 0.05)
- **MD**: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
- **LG**: 0 10px 15px -3px rgba(0, 0, 0, 0.1)
- **XL**: 0 20px 25px -5px rgba(0, 0, 0, 0.1)

## Component Guidelines

### Buttons
- **Primary**: Blue background, white text, medium padding
- **Secondary**: Gray border, gray text, medium padding
- **Danger**: Red background, white text, medium padding
- **Icon**: Square, icon only, minimal padding

### Cards
- White background, subtle shadow, rounded corners
- Consistent padding (16px-24px)
- Clear hierarchy with titles and content

### Navigation
- Sidebar: Dark theme with blue accents
- Top bar: White background with user actions
- Breadcrumbs: Gray text with blue links

### Forms
- Clear labels above inputs
- Consistent input heights (40px-48px)
- Error states with red borders and text
- Success states with green borders

### Tables
- Alternating row colors for readability
- Sortable headers with hover states
- Action buttons in the last column
- Responsive design for mobile

### Charts
- Consistent color scheme matching brand
- Clear legends and labels
- Interactive tooltips
- Responsive sizing
